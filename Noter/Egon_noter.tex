\section{Strukturovervejelser}

\subsection{Hvad Thomas sandsynligvis mente}

\begin{itemize}
    \item \textbf{ARIMA} $\rightarrow$ baseline på dansk inflation
    \item \textbf{GETS $\rightarrow$ Dynamic regression eller VAR} med dansk arbejdsløshed på dansk inflation. Det er den klassiske Phillips curve-test: har arbejdsløshed forecasting-værdi for inflation?
    \item \textbf{Granger-kausalitet med EU-data} som et \textit{analytisk element} -- altså ikke en tredje model, men en test der besvarer: er dansk inflation påvirket af europæiske forhold? Det bruges til at \textit{motivere} eller \textit{diskutere} modellen, ikke som en selvstændig forecasting-model.
\end{itemize}

\subsection{Hvad det betyder for den tidligere anbefaling}

Den tidligere anbefaling var ikke så langt fra, men EU-delen blev overkompliceret ved at smide Eurozone-inflation ind i selve VAR-modellen. Det er nok for meget.

En renere og skarpere struktur ville være:

\begin{enumerate}
    \item \textbf{ARIMA} -- univariat baseline på dansk inflation
    \item \textbf{GETS $\rightarrow$ Dynamic regression eller VAR} -- dansk inflation og dansk arbejdsløshed (Phillips curve)
    \item \textbf{Granger-kausalitet} som et diskussionselement: påvirker EU-inflation dansk inflation? Det kan bruges i data-afsnittet eller i diskussionen til at kontekstualisere resultaterne
\end{enumerate}

Det er en mere fokuseret opgave, og man undgår at drukne i for mange variable.

\section{Research Question}

\textbf{Does the Phillips Curve have forecasting value for Danish inflation,
and does incorporating unemployment data improve forecasts relative to a
univariate benchmark?}

The Phillips Curve describes the empirical relationship between inflation
and unemployment. From a forecasting perspective it raises a concrete and
testable question: can a model that also uses unemployment outperform a
pure time-series model of inflation? This is the central question of the
assignment.

The relationship is also known to be \textbf{unstable over time}. The
Phillips Curve ``disappeared'' in many countries during the 1980s and
1990s, and there is ongoing debate about whether it has re-emerged
post-COVID. Structural breaks are therefore a core part of the analysis.
Key references are \textcite{stockwatson1999} and \textcite{galigertler1999}.

\section{Methods}

Det centrale spørgsmål er om en model der inkorporerer arbejdsløshed kan
slå en univariat ARIMA-benchmark. Analysen bygges op gennem følgende stages.

\subsection{Stage 1: Exploratory Analysis}

\begin{itemize}[nosep]
    \item Tidsplots for alle serier med visuel inspektion for trend,
          strukturelle skift og usædvanlige observationer.
    \item ACF og PACF plots til at karakterisere autokorrelationsstruktur.
    \item ADF og KPSS unit root tests til at bestemme integrationsorden.
\end{itemize}

\subsection{Stage 2: Univariate ARIMA (Baseline)}

Den primære benchmark som den multivariate model skal slå for at
demonstrere at arbejdsløshed indeholder forecasting-information ud over
hvad der allerede ligger i tidligere inflationsværdier.

\begin{itemize}[nosep]
    \item Fit ARIMA$(p,d,q)$ til dansk inflation via Box-Jenkins proceduren.
    \item Vælg final specifikation via AIC og BIC.
    \item Residualdiagnostik: Ljung-Box test for autokorrelation.
\end{itemize}

\subsection{Stage 3: General-to-Specific Model Selection (GETS)}

GETS er særligt velegnet her fordi kandidatsættet af regressorer er bredt:
dansk inflation, dansk arbejdsløshed og laggede værdier af begge. Ved at
starte bredt og teste ned finder vi den parsimonious model data faktisk
understøtter, frem for at påtvinge en specifikation på forhånd. Dette er
tilgangen anbefalet af \textcite{doornik2018} og et metodisk element
forelæseren specifikt fremhævede som underbrugt.

\begin{itemize}[nosep]
    \item Specificer et General Unrestricted Model (GUM):
          \[
          \pi_t^{DK} = \alpha
            + \sum_{i=1}^{p} \beta_i \pi_{t-i}^{DK}
            + \sum_{j=0}^{q} \gamma_j u_{t-j}^{DK}
            + \varepsilon_t
          \]
          med initial laglængde sat generøst ($p = q = 4$).
    \item Anvend sekventielle $t$-tests og $F$-tests til at eliminere
          insignifikante regressorer.
    \item AIC og BIC som krydstjek på final specifikation.
\end{itemize}

\subsection{Stage 4: Model 2 -- Dynamic Regression eller VAR}

\textit{Vælg én af de to følgende modeller baseret på hvad data og
analysen peger på.}

\subsubsection{Option A: Dynamic Regression (ARDL)}

\begin{itemize}[nosep]
    \item Estimer den parsimonious model fra Stage 3 som en dynamic
          regression med arbejdsløshed som ekstern regressor.
    \item Inkorporer strukturelle breaks som dummyvariable hvor relevant.
    \item Residualdiagnostik: Ljung-Box og ARCH test.
\end{itemize}

\subsubsection{Option B: Vector Autoregression (VAR)}

\begin{itemize}[nosep]
    \item Specificer en bivariat VAR i $(\pi_t^{DK},\, u_t^{DK})$.
    \item Vælg laglængde via AIC / BIC.
    \item Granger kausalitetstest: Granger-causes arbejdsløshed inflation?
    \item Impulse response functions til at trace dynamikken.
\end{itemize}

\subsection{Stage 5: Granger-kausalitet med EU-data (diskussionselement)}

Uanset valget i Stage 4 anvendes Granger-kausalitet med EU-data som et
analytisk element til at kontekstualisere resultaterne:

\begin{itemize}[nosep]
    \item Granger-causes EU-inflation dansk inflation?
    \item Bruges til at motivere eller diskutere modellen -- ikke som en
          selvstændig forecasting-model.
\end{itemize}

\subsection{Stage 6: Forecast Evaluation}

\begin{itemize}[nosep]
    \item Hold de sidste 12--24 måneder tilbage som evalueringsperiode.
    \item Generer forecasts fra alle modeller med et rolling window.
    \item Sammenlign præcision via RMSE og MAE.
    \item Rapporter 95\% prediction intervals og vurder empirisk dækning.
    \item Reflekter over forecast-horisonten: usikkerheden stiger hurtigt
          jo længere ud i fremtiden man forecaster.
\end{itemize}
